<!-- {Provide a description of your app.  Describe how to use the app.} -->

This app is a conversation starter app. This is very proof-of-concept at the moment. Currently, You are presented with a random conversation starter question. You then have two buttons. One button lets you skip it and come back later. The other button lets you mark it as answered and it won't show it again (until restarting the app, at least for now). I will give you a warning though: While I have the ability to fix this, there is still a bug where if you answer all the questions the app will be stuck in an infinite loop. So don't use it too much lol.

<!-- {Describe your purpose for creating this app.} -->

I created this app for two main reasons. The first reason, is that I wanted to learn how to make a decent app. The second, is that my SO and I have been wanting a good way to handle conversation cards for a while, and this seemed like a good way to do it.
